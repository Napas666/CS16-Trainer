# CS 1.6 Trainer

External trainer for Counter-Strike 1.6 (Steam, `hl.exe`).  
Учебный проект — только для игры с ботами оффлайн.

## Возможности

| Модуль | Описание |
|--------|----------|
| 🎯 **AIM** | Плавное наведение на врагов, настройка FOV и smooth, прицел в голову / тело |
| 👁 **ESP / WH** | Цветные боксы поверх игры, HP-бары, snaplines |
| 🔫 **PEN / PenBox** | Зелёный бокс = враг за простреливаемой поверхностью (металл/дерево < 500 ед.) |
| 🐇 **BHOP** | Авто-прыжок в момент приземления |
| ✚ **Прицел** | 5 стилей, настройка цвета / размера / gap / толщины / точки |

## Запуск

```
1. Запусти CS 1.6 → зайди на карту с ботами
2. Запусти RUN.bat от имени администратора
3. Нажми ATTACH TO CS 1.6
4. Включай нужные модули в табах
```

## Как работает (кратко)

- **Memory reading** — `pymem` читает `hl.exe` через `ReadProcessMemory`
- **Viewangles** — запись в `hw.dll + 0x6E3440` поворачивает камеру
- **Entity list** — `client.dll + 0x1136F8C` содержит массив `cl_entity_t` с позициями игроков
- **Overlay** — `pygame` + `WS_EX_LAYERED | WS_EX_TRANSPARENT` = прозрачное окно поверх CS

> Если оффсеты не работают — открой Cheat Engine, найди своё HP, и пересчитай базу `client.dll`.
